# DodgeBall
- A simple 2D dodge game development project

## Project Structure

```
dodge-ball/
├── Application/     # Game logic (scenes, actors, etc.)
├── Framework/       # Game framework (interfaces and manager systems)
├── DataPack/        # Game data packaging tool
├── Config/          # Configuration files (CoreConfig.yaml, GameConfig.yaml)
├── Tool/            # Build/data automation scripts (batch/Python)
└── ThirdParty/      # External libraries
```
